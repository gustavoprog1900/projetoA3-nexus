# 🎮 Nexus Gaming Store

Nexus Gaming Store é uma landing page de e-commerce gamer desenvolvida utilizando apenas HTML5, CSS3 e JavaScript Vanilla.

O projeto foi criado com foco em um design moderno, responsivo e intuitivo, apresentando uma interface elegante em tons de preto e cinza para destacar produtos e melhorar a experiência do usuário.

## 🚀 Funcionalidades

- Header responsivo
- Menu mobile
- Hero Section moderna
- Estatísticas da loja
- Filtro de categorias
- Exibição de produtos
- Cards com efeito hover
- Seção de benefícios
- Newsletter com validação
- Botão "Voltar ao topo"
- Design responsivo para dispositivos móveis

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome

## 📱 Responsividade

O projeto foi desenvolvido para funcionar corretamente em:

- Desktop
- Notebook
- Tablet
- Smartphones

O objetivo deste projeto é demonstrar conhecimentos em desenvolvimento Front-End, criação de interfaces modernas e construção de layouts responsivos para aplicações web.
